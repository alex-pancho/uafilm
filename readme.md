# Ukrainian online films parser
###### based on Python framework Scrapy

### parser_name: site
* ukino: uakino.club
* eneyida: eneyida.tv

### Prepared
1. Download and extract
1. Create virtual environment 
   `python -m venv`   
1. Install requirements 
   `pip install -r requirements.txt`
   
### Run

`cd uakino`

`scrapy crawl parser_name`
